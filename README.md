# Engram

A lightweight system for capturing inputs, consolidating knowledge, generating ideas executing work, and preserving historical record.

## Cognitive schema

Knowledge is organized into five primary types:

```
/
  research/  # raw inputs, exploration, source material, generated reports
  knowledge/ # durable concepts, distilled understanding, atomic mental models
  ideas/     # speculative directions, new hypothesis, partial thoughts
  tasks/     # committed, actionable work
  journal/   # chronological record of events, outcomes, observations, and reflection
```

## Core concepts

Capture lightly, consolidate deliberately, execute narrowly, reflect regularly.

This framework operates through three loops:

* Input → ingest, explore, capture
* Processing → structure, connect, decide
* Output → execute, reflect, adapt

And mirrors how the brain works:

* Most inputs are discarded
* Some are encoded
* Very few shape behavior

## Cognitive routines (Agentic skills)

### Daily view

A lightweight digest that highlights:

* Today's tasks
* Fresh ideas
* New research

### Consolidation process

Regular processes distill raw inputs and experiences into durable knowledge and future actions.

* Daily
  - reconcile tasks
  - log journal
  - queue research
  - promote if obvious

* Weekly
  - summarize completed tasks
  - promote research and ideas
  - prioritize upcoming tasks
